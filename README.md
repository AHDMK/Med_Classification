# Medical Text Classification

AI/NLP internship project focused on classifying medical text into predefined categories.

## What I did
- Prepared and cleaned domain-specific medical text data
- Built and evaluated classification pipelines
- Tested ML/DL models including SVM, CNN, LSTM, CNN-LSTM, attention models, and LLM-based methods
- Fine-tuned language models for specialized tasks
- Created labeled evaluation datasets for better reliability

## Results
Reached 90–97% accuracy on target classification tasks.
